# 7. `hooks/`

Custom React hooks.

```
hooks
 в”њв”Ђв”Ђ useAuth.js
 в”њв”Ђв”Ђ useFetch.js
 в””в”Ђв”Ђ useLocalStorage.js
```

Example:

```javascript
import { useState, useEffect } from "react";

export function useFetch(apiCall) {
  const [data, setData] = useState([]);

  useEffect(() => {
    apiCall().then(res => setData(res.data));
  }, []);

  return data;
}
```
